The company's frontend projects fall into five main types:

2. **Admin dashboards** (e.g., `epcm`, `tism`, etc.)
3. **Heavyweight enterprise-level admin systems** (e.g., `sbom`, `eplm`, etc.)
4. **Mobile H5 applications**
5. **WeChat Mini Programs**
